## 🧰 Installation

Clone this project

```bash
  git clone https://github.com/alimprasetyo77/FlixTv.git
  cd FlixTv
```

#### ⌛ Install

```bash
  npm install
```

#### 🚀 Usage

```bash
  npm run dev
```
